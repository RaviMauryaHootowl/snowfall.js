# Snowfall.js

Nice snow in a JS Canvas.


## Instructions

First, install.

```sh
yarn install snow-fall
```

Then...

```html
<!-- You are expected to size and position this yourself with CSS -->
<div id="snow-container"></div>
```

```js
const snowfall = require('@erikwatson/snowfall')
snowfall()
```

## Try it out

  + git clone
  + nmp/yarn install
  + npm/yarn run watch
